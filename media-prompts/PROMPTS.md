# Gemini video prompts — one per slot

Generate each clip (Gemini video / Google Flow), then save it into
`public/media/clips/` with **exactly the filename shown** and redeploy.
The site detects the file automatically — no code changes.

Shared style rules baked into every prompt: neon anime aesthetic, deep navy
background `#0d0f1f`, hot pink `#ff4fa3` + electric cyan `#4fe3ff` + violet
`#8b5cf6` + gold `#ffe14d` accents, stylized silhouettes only — never a
photoreal person, no faces, no text, no watermark, seamless loop.

Keep files small: H.264 MP4, target ≤ 5 MB (a `.webm` sibling with the same
name is tried first if you add one).
